# mecab_dictionaries

create various dictionaries for MeCab and MeCab CLI using fugashi.

following libraries, commands and apps are needed.

* Libraries
    * gcc (g++)
    * Python3
* Commands
    * patch
    * make
    * wget
    * git
    * zip (unzip)
    * bzip2 (unbzip2)
    * xz (unxz)
    * nkf
    * sed (GNU version)
* Applications
    * poetry
